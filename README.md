# QQ Farm CDP Manager - Bernadette Edition

由 [bernadettesh38-bot](https://github.com/bernadettesh38-bot) 维护的 QQ 经典农场自动化工具。

**公益产品，免费分享，禁止倒卖。**

当前版本：`1.8.5`

## 本版更新

- 修复微信农场窗口后台运行后消失、恢复成极小窗口以及放大后画面模糊的问题；现在窗口保持原状态，通过轻量运行时脉冲支持被电影或直播窗口覆盖时继续工作。
- 新增旧版 `farm-config.json` 一键导入，导入前自动备份现有配置，受保护版同样可用。
- 桌面壳和 Web 项目说明已改用 QQ 频道“种点田喝点茶吧”二维码。

## 现有功能

### 自家农场

- 自动收获、浇水、除草、杀虫和枯萎清理
- 分批催熟快速追收与收获后优先补种
- 单地块及批量铲除
- 土地状态、成熟倒计时、变异与多季作物识别

### 好友农场

- 好友列表刷新、进场、偷菜和可选帮忙
- 白名单、黑名单、作物黑名单和访问冷却
- 好友逐块催熟复扫、短时追踪和定向优先复访
- 琉璃宝荷等多格作物动态锚点识别与定点收取

### 种植与施肥

- 背包种子读取、指定种子及多种自动种植策略
- 普通肥、有机肥识别、批量施肥和催熟联动收获
- 多季补肥、连续催熟及施肥状态跟踪

### 运行与界面

- QQ Lite 与微信/CDP 两条运行路线
- 串行任务调度、运行时自动恢复和异地登录重连
- Web 控制台、桌面壳、统计记录与消息推送
- GitHub 入口和 QQ 频道意见反馈二维码

## 快速开始

### 环境要求

- Node.js 22 或更高版本
- QQ 路线：Windows / macOS
- 微信路线：Windows，并具备微信 PC 小程序调试环境

### Windows

双击：

```text
Windows_start.bat
```

按提示选择普通启动或 Lite 模式，以及 QQ 或微信路线。默认控制台地址：

```text
http://127.0.0.1:8787/
```

### 命令行

```bash
npm install

# QQ
npm run start -- --qq

# 微信
npm run start -- --wx
```

## 多格作物偷菜

- 支持琉璃宝荷等四格作物，不依赖固定地块编号或固定种植位置。
- 进入好友农场后会根据实时占位数据，通过锚点地块 ID、节点路径或网格坐标动态定位真实植物。
- 四个覆盖格会归并为一个收获目标，只点击真实主锚点，避免重复点击、点击覆盖格或漏偷。
- 当好友列表显示可偷，但场内 `canSteal` 标记缺失时，仍会结合成熟状态和果实数量执行兜底收取。

## 更新 Lite 包

修改 `button.js` 后运行：

```bash
npm run qq:build-lite
```

生成文件必须保持在项目设置的 `210 KB` 上限以内。
